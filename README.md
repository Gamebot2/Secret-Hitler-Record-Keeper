# Secret-Hitler-Record-Keeper
Web application to keep track of player records and statistics for the popular board game Secret Hitler.

## Built With

* [AngularJS](https://angular.io/) - Web Framework
* [Maven](https://maven.apache.org/) - Dependency Management
* [MySQL](https://www.mysql.com/) - Database Management
